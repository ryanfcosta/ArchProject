# ArchProject
## Proposta
A proposta do projeto é trazer de forma visual e educacional o funcionamento da microarquitetura MIC-1 e do RISC-V
Para facilitar a representação, os subciclos serão abstraídos, focando então na exibição dos ciclos que são executados dentro de cada instrução.

## Estudos
Foram utilizados livros sobre arquiteturas, websites e arquivos sobre MIC 1 e RISC-V e um simulador de MIC 1 para estudos e testes sobre os datapaths e as instruções utilizadas no projeto.

## Perspectiva de Features
Temos o objetivo de, durante o desenvolvimento do projeto  
- Integrar mais de um processador (MIC-1 e MIC-1 ou MIC-1 e RISC-V)
- Simplificar a linguagem de comandos e instruções
